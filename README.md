# -GuessGameDemoInSwift5
Swift5, 简单猜猜游戏 Demo, 缺少组件封装, 能跑就行, 还要什么自行车, 不是吗?

## Features - 特性
- 题目板块有毛玻璃效果 (.blur 实现)
- 进度条有动效哟
- 小编实在编不出来了, 就酱叭

## Version 1.0
- 加入背景云彩动效

## Why
- 学习 Swift 过程中需要有自己输出的过程, 所以自己做了这么一个小东西

## Usage
- 只需要拿去改一下颜色, 改一下 QuestionDataModel 里面的数据就可以成为一个猜猜小游戏了
